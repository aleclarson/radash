## Basic usage

Returns the mapped value for the first element that satisfies the specified condition--else undefined.
If the filter is omitted, returns the first non-nullish mapped value.

```ts
import * as _ from 'radashi'

const fish = [
  {
    name: 'Marlin',
    weight: 105,
    source: 'ocean',
  },
  {
    name: 'Bass',
    weight: 8,
    source: 'lake',
  },
  {
    name: 'Trout',
    weight: 13,
    source: 'lake',
  },
]

_.selectFirst(
  fish,
  f => f.weight,
  f => f.source === 'lake',
) // => 8
```
