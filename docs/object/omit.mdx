## Basic usage

Given an object and a list of keys in the object, returns a new object without any of the given keys.

```ts
import * as _ from 'radashi'

const fish = {
  name: 'Bass',
  weight: 8,
  source: 'lake',
  brackish: false,
}

_.omit(fish, ['name', 'source']) // => { weight, brackish }
```
